# 🏡 Airbnb Clone

An Airbnb-inspired property listing and search app built with Next.js 14 and Mapbox — featuring interactive map-based search and a TypeScript-powered frontend.

## ✨ Features

- **Interactive Map** — Explore listings on a live Mapbox map with location markers
- **Property Listings** — Browse and filter available properties
- **Search & Filters** — Filter by location, dates, and guest count
- **Responsive UI** — Mobile-first design built with Tailwind CSS
- **TypeScript** — Full type safety across the entire codebase

---

## 🛠️ Tech Stack

Next.js 14 · TypeScript · Mapbox GL JS · Tailwind CSS

---

## 🚀 Getting Started

```bash
git clone https://github.com/mohamed-ayad40/Airbnb-clone.git
cd Airbnb-clone
npm install
# Add NEXT_PUBLIC_MAPBOX_TOKEN to .env.local
npm run dev
```
